Ticket Types

General Ticket — support inquiries, moderator assistance, or reporting issues.

Roster Ticket — application channel for users seeking to join Void Esports.

Protocol

All ticket responses must begin with a professional greeting and age inquiry.

Roster Categories

  • Pro Roster — 25,000+ Power Ranking; $1,000+ Fortnite earnings
  • Semi-Pro — 10,000-25,000 PR
  • Academy Roster — 500-10,000 PR
  • Creative Roster — Smooth mechanics; Provide clips
  • Streamer — 1k followers; 25+ viewers
  • Content Creator — 1k followers; 10k TikTok
  • Junior Content Creator — 500 followers; 3k TikTok
  • GFX/VFX — Portfolio required; High-quality work

Mod Commands & Expectations

Commands: warn, report, loa

Weekly expectations: 10 tickets; 400 messages

Ticket Handling

Professional greeting required. Examples: "Hello, how may I help you?" Age verification: "May I ask how old you are?"

Role Responses

Creative applicants → request clips. Academy → request tracker. Pro/Semi-Pro → tracker + default dance. Content creators → social links. GFX/VFX → portfolio. Coach → ask for player trackers.

Void Grinder+

If they do not qualify for rosters: require creator code screenshot, username containing "Void", Discord tag.

Blacklist Format

User: Username
User ID: Discord ID
Reason: Underage

Closing Script

Perfect! Please change your name to include "Void" and send a screenshot of using our code. Once verified: Thank you! Here are your roles! Have a nice day!
